Powers/Abilities:
Altered String
Kushina has the ability to create and manipulate strings at will, the strings themselves are razor sharp and are able to cut through objects if enough energy is put through them, however, they can be cut. The strings know no friend nor foe as they can cut any person. The user is even able to be cut by the strings, but only if they allow the strings to do so. The strings can be attached to people (with permission) and deceased bodies in order to make them act as though they were moving normally. The strings can be run through the body and act as stitching for external and internal body injuries, allowing them to sow up a wound or whatever.

String Type Two
This second type of string is an extension of the first type of string. This allows the user to do two additional things.

Thirst
The first thing is that the strings are able to consume any blood that comes in contact with the strings, this automatically causes the person's blood that came in contact with the string to be a target. People who become targets automatically have a lowered defense against the strings and are thus, more at risk of getting cut up. The only way to make a person a target for the Thirst to start is for the user to flick a finger at the target and have a piece of string shoot off and pierce the target's body. 'Bullet Seed' 2 Post CD

Restoration
This powers up the strings repairing ability to that of a limited reconstruction. It allows the strings to not only connect body parts back together, it also allows for restoration of parts that cannot be normally sown back, such as the eyes, tendons, and muscles to an extent. 2 Post CD; 3 Post For Completion

Latching Onto You
The strings that are set up are able to connect with and cut immaterial or ethereal objects that can not be touched by the human hand such as clouds, light, and etc. However, this can only be on for a certain amount of time. 3 Post Duration; 3 Post CD
